## Usage

1. **Adjust Settings**:
   - Select the number of phone numbers to generate (1-50)
   - Optionally specify an area code
   - Choose your preferred format

2. **Generate Numbers**:
   - Click "Generate Numbers" to create unique phone numbers
   - Each number is guaranteed to be unique across all users

3. **Copy Numbers**:
   - Copy individual numbers or all numbers at once
   - Use for testing, development, or demonstration purposes
